Maintenance
-----------
The site was generated by a Python script collection in the build
directory of the source environment. The shared library lib.py
holds the head, navigation, footprint, footer and JSON-LD scaffolding;
each phase script (phase1_tech, phase1_core, phase1_catalogue,
phase1_reference, phase1_retrofit, phase2_blog, phase3_integrity)
produces a related set of pages by importing from lib.py. To rebuild
the entire site, run the phase scripts in order.
